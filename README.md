# 西方艺术史思维导图

一个交互式的西方艺术史思维导图应用，涵盖从史前艺术到现代艺术的完整发展脉络。

## 在线演示

访问地址：https://againster1992-debug.github.io/Art-History-Mindmap/

## 功能特点

- **思维导图可视化**：1277 个节点，10 个层级，清晰展示艺术史发展脉络
- **交互式探索**：点击节点查看详细介绍、图片、相关链接
- **多层级展开**：支持折叠/展开，自由探索不同层级的细节
- **标签系统**：每个节点配有标签（流派、时期、艺术家国籍等）
- **图片画廊**：支持多图浏览，底部滚动条切换
- **莫兰迪配色**：不同层级采用莫兰迪色系，视觉舒适
- **海獭彩蛋**：右下角点击海獭跳转作者知乎

## 内容覆盖

- 史前艺术（初民的艺术）
- 古代艺术（埃及、希腊、罗马）
- 中世纪艺术（拜占庭、罗马式、哥特式）
- 文艺复兴
- 巴洛克与洛可可
- 19 世纪艺术（新古典主义、浪漫主义、现实主义、印象派）
- 现代艺术（后印象派、表现主义、立体主义、超现实主义等）
- 当代艺术

## 技术栈

- 纯 HTML/CSS/JavaScript 单文件应用
- SVG 矢量绘图
- 响应式设计
